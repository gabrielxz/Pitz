# Legacy

The history behind the remake.

## `c-original/` — the real first game (2006–2007)

The original Pitz, written in C with SDL + OpenGL. It grew over four versions:

- **0.1–0.3** — the grid, movement, the lights mechanic, and the follow-spotlight.
- **0.4** (April 2007) — textured grass and pits, a splash screen, and the
  stick-figure player the dev notes call "Mr SquareHead." This is the version
  the remake is based on.

Build artifacts (`.o`, `.exe`) are not tracked. The grass/pit textures here
(`*.ppm`) are the same ones used by the remake (converted to PNG under
`public/textures/`).

## `unity-stub/` — an abandoned restart (2018)

A Unity 2018 project that was going to be a remake but never got past a
Hello-World scene. Kept for the record; not used by the remake.
